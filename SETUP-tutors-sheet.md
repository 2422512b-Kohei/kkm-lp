# 講師検索ページ（tutors.html）とGoogleスプレッドシートの接続手順（B案）

講師用Googleフォームの回答スプレッドシートを、そのまま `tutors.html` の検索対象にできます。
**IDを1つ貼るだけ**で、以降はスプレッドシートを更新すればサイトに自動反映されます。

---

## 1. スプレッドシートを用意する

1. 講師用Googleフォームの回答先スプレッドシートを開く（または新規作成）
2. **1行目（ヘッダー）** を下の「列名」に合わせる
   - フォームの質問文をそのまま使う場合は、別シートを作って `=` で参照し、
     ヘッダーだけ下記の列名に整える方法がおすすめです

### 列名（1行目）と入力する値

| 列名 | 入力する値 | 例 |
|---|---|---|
| `公開` | 掲載する行だけ `TRUE`／`公開`／`○` | TRUE |
| `ニックネーム` | 公開用の呼び名（**本名は入れない**） | たく先輩 |
| `学部学年` | 学部・学年 | 神戸大 工学部 2年 |
| `評価` | 10基準の平均（**4.0以上のみ掲載**されます） | 4.2 |
| `指導科目` | 指導できる科目 | 数学・理科 |
| `コメント` | 語り口調の一言 | 田舎の公立校から独学で… |
| `出身地` | `local` / `city` / `transfer` | local |
| `きょうだい` | `first` / `middle` / `last` / `only` | first |
| `幼少期` | `lessons` / `free` | free |
| `学校歴` | `public` / `jhs` / `hs`（複数可） | public, hs |
| `学習スタイル` | `self` / `cram` | self |
| `得意不得意` | `effort` / `inquiry` | effort |
| `関わり方` | `companion` / `leader` | companion |
| `コミュニケーション` | `empathy` / `logical` | empathy |
| `やる気スイッチ` | `competition` / `curiosity` / `safety`（複数可） | safety, curiosity |

> **複数可** の列は、カンマ・読点・スペース・スラッシュ区切りでOK（例：`public, hs` / `安全 好奇心`）。

### 値の対応表（何を選ぶ？）

- **出身地**：`local`=地方出身／`city`=都市部出身／`transfer`=転勤族・引っ越し経験あり
- **きょうだい**：`first`=長子／`middle`=中間子／`last`=末っ子／`only`=一人っ子
- **幼少期**：`lessons`=習い事多め派／`free`=のびのび派
- **学校歴**：`public`=公立中心／`jhs`=中学受験経験あり／`hs`=高校受験経験あり
- **学習スタイル**：`self`=独学・工夫型／`cram`=塾・予備校活用型
- **得意不得意**：`effort`=努力型／`inquiry`=探求型
- **関わり方**：`companion`=伴走型／`leader`=牽引型
- **コミュニケーション**：`empathy`=じっくり共感・ほめて伸ばす／`logical`=論理的・的確に課題解決
- **やる気スイッチ**：`competition`=競争・達成感／`curiosity`=好奇心・面白さ／`safety`=心理的安全性・安心感

> フォームの選択肢を上記の日本語ラベルで作り、スプレッドシート側で
> `=SWITCH(...)` などで英字コードに変換しておくと、入力者は日本語のまま選べます。

---

## 2. スプレッドシートを「閲覧可能」に共有する

1. 右上の **「共有」** をクリック
2. 「一般的なアクセス」を **「リンクを知っている全員」** に変更
3. 権限は **「閲覧者」** のままでOK（編集権限は不要）

> これで、ページ側（ブラウザ）からデータを読み取れるようになります。
> 公開されるのは、`公開=TRUE` かつ `評価≧4.0` の行の、上表の列だけです。

---

## 3. スプレッドシートIDをコードに貼る

スプレッドシートのURLを見ます：

```
https://docs.google.com/spreadsheets/d/【ここがID】/edit#gid=0
```

`tutors.html` の上部にある設定を書き換えます：

```js
var CONFIG = {
  SHEET_ID: 'ここに貼る',   // 例: 1AbCdEf... 
  SHEET_GID: '0',          // 対象タブのgid（URLの #gid= の値）
};
```

保存してデプロイすれば完了です。ページ上部の帯が
**「最新の認定講師 ● 名を表示中」** に変われば接続成功です。

---

## 動作の仕組み（かんたん）

- ページを開くたびに、スプレッドシートの内容を読み込みます（サーバー不要）
- `公開=TRUE`・`評価≧4.0` の行だけを表示（＝**行の選抜**）
- 上表の列だけを使用（本名・連絡先などは読み込まない＝**列の選抜**）
- ご家庭が選んだ条件と、各先生のタグの一致数を数えて、**多い順に並び替え**

## うまく表示されないとき

- 帯が赤（読み込み失敗）→ 共有設定が「リンクを知っている全員／閲覧者」になっているか確認
- 先生が出てこない → `公開` 列が `TRUE` か、`評価` が 4.0 以上かを確認
- タグが反映されない → 値が英字コード（`local` など）になっているか確認

---

## プライバシー面の注意

- 公開してよい項目だけをシートに置く（本名・住所・連絡先・生の志望理由などは別シートで管理）
- フォームに「**匿名プロフィールの公開に同意する**」チェック欄を用意しておく
