# インターン日程かぶりチェッカー

複数のインターン候補日程をテキストから一括抽出し、カレンダー上で「日程の被り（バッティング）」を視覚的にあぶり出すシングルページ Web アプリ（SPA）です。

外部 API は一切使用せず、フロントエンド（Next.js / TypeScript）のロジックだけで完結します。

## 技術スタック

- **Next.js 14** (App Router, TypeScript)
- **Tailwind CSS 3**
- **lucide-react**（アイコン）

## セットアップ

```bash
npm install
npm run dev      # http://localhost:3000 で開発サーバー起動
```

本番ビルド:

```bash
npm run build
npm run start
```

## 主な機能

1. **テキスト解析（正規表現ベース）**
   `7月21日(火)` / `8/4` などの書式から月日を一括抽出。企業名は専用入力欄、または入力テキストの 1 行目から自動取得。年は 2026 年固定。
2. **カスタムカレンダー**
   Tailwind のグリッドで自作。7〜12 月など月を自由に切り替え可能。
3. **被り（バッティング）の可視化**
   同じ日に 2 社以上重なるとセルが薄い赤（`bg-red-50`）になり「⚠️ N社」ラベルを表示。3 社以上はさらに濃く（`bg-red-100`）。
4. **企業ごとのサイドバー管理**
   企業一覧をパステルカラー付きで表示。チェック（目アイコン）で表示/非表示トグル、削除ボタン付き。
5. **データ永続化**
   登録データは `localStorage` に自動保存され、リロードしても消えません。

## ディレクトリ構成

```
src/
├─ app/
│  ├─ layout.tsx       # ルートレイアウト
│  ├─ page.tsx         # メイン画面（状態管理・localStorage）
│  └─ globals.css      # Tailwind エントリ
├─ components/
│  ├─ InputForm.tsx    # 日程入力フォーム + 抽出プレビュー
│  ├─ Sidebar.tsx      # 企業一覧・表示切替・削除
│  └─ Calendar.tsx     # 月切替カレンダー・重複ハイライト
└─ lib/
   ├─ parse.ts         # テキスト → 月日 抽出ロジック
   ├─ colors.ts        # パステルカラーパレット
   └─ types.ts         # 型定義
```
