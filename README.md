# KKM｜神戸大生専門 家庭教師紹介サービス LP

神戸大生に特化した家庭教師の直接マッチングサービス「KKM（仮称）」の、ご家庭向けランディングページです。

## 概要

- **対象**: お子様の家庭教師を探しているご家庭
- **特徴**: 神戸大生限定・事前面談済みの認定講師を紹介。教材費・管理費は0円、初回のマッチング手数料のみ。
- **構成**: スマートフォン表示を最優先した、1ページ完結のレスポンシブLP。

## ファイル

| ファイル | 説明 |
|---|---|
| `index.html` | LP本体。HTML・CSS・JavaScript を1ファイルに同梱。 |

## 使用技術

- HTML5 / CSS3
- [Tailwind CSS](https://tailwindcss.com/)（CDN）
- [Font Awesome](https://fontawesome.com/)（CDN・アイコン）
- JavaScript（認定基準のアコーディオン開閉、スクロール表示）

外部ライブラリはすべてCDN経由のため、ビルド作業は不要です。

## ローカルで表示する

`index.html` をブラウザで開くだけで表示できます。

```sh
# Windows の場合
start index.html
```
